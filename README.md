## Divide and Conquer
### Find Maximum Subarray
1. Brute-force approach: O(n^2)
2. Divide and conquer approach: O(n\*lg(n))
3. Linear-time approach: O(n)

##Sort

3. Best case: O(n\*lg(n))

###Quicksort

3. Best case: O(n\*lg(n))
